# Fitness Wrapped

Spotify Wrapped–inspired fitness analytics platform that transforms Apple Health and Strava data into animated, story-driven visualizations using Next.js, TypeScript, and FastAPI.

## Features

- Apple Health + Strava integration
- Animated story-based slide engine
- Dynamic route visualization rendering
- Scroll-snapped immersive UI
- Responsive mobile-first design
- Interactive yearly analytics dashboard

---

## Tech Stack

- Next.js
- TypeScript
- FastAPI
- React
- CSS Animations
- Apple Health Data
- Strava API

---

## Architecture

Apple Health / Strava → FastAPI Backend → Data Processing → Wrapped Generator → Next.js Frontend

---

## Future Improvements

- AI-generated fitness insights
- Friend comparisons
- Social sharing
- Native iOS app
- Cloud sync

## Run Locally

```bash
npm install
npm run dev
```

Backend:

```bash
uvicorn main:app --reload
```
